## Installation

```sh
npm i -D bs-reach-ui
# or yarn
yarn add bs-reach-ui
```

* Add `bs-reach-ui` in `bsconfig.json`
* Add needed CSS style in your application (see https://ui.reach.tech/styling)

## Run Project

* Install dependencies

```sh
npm install
# or yarn
yarn
```

* Run

```sh
npm start
```

## Run examples

```sh
npm run examples
# or yarn
yarn run examples
```
